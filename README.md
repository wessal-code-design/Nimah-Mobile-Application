# Nimah Mobile Application

The Nimah Project embodies the spirit of nourishment, fostering a harmonious balance between culinary excellence and social responsibility. It seamlessly blends the art of cooking with the spirit of caring for others — serving more than just food.

Designed specifically for the Saudi market, Nimah reflects local values, cultural context, and user behaviors, delivering a meaningful digital experience that supports community engagement and responsible food practices.

This project focuses on creating an intuitive, human-centered mobile application that prioritizes usability, accessibility, and emotional connection through thoughtful UI/UX design.

## Project Overview
- Platform: Mobile Application (iOS / Android)
- Target Market: Saudi Arabia
- Focus: UI/UX Design
- Design Approach: User-Centered & Social Impact–Driven
- Tools Used: Figma, Miro, Whimsical, Photoshop, Illustrator

## Design Process
- User research & market analysis (Saudi audience)
- User personas & user journeys
- Information architecture & user flows
- Wireframing
- High-fidelity UI design
- Prototyping & interaction design
